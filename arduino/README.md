## Arduino Message Listener

### Communication

You can connect to it via UART over USB, with baud rate 9600. 

### Message Format

```
<Message1>,<Message2>
```

Obviously, limited to LCD display size.

### Hardware

- Arduino Uno,
- 16x2 LCD display,
- breadboard, potentiometer and jumper wires.
